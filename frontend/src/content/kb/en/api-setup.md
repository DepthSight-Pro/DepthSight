# API Keys Setup

In order for DepthSight to trade on your behalf or receive balance data, you need to connect your exchange API keys.

## Safety First

We recommend using **only** keys with read and trade permissions. **Never** enable withdrawal permissions.

### Connection Steps:

1. Go to **Settings** -> **API Keys**.
2. Click the **Add Key** button.
3. Select your exchange (e.g., Binance or Bybit).
4. Enter the API Key and API Secret.
5. Test the connection using the **Check** button.

If everything is configured correctly, the key status will change to "Active".
