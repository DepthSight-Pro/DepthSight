# Настройка API ключей

Для того чтобы DepthSight мог торговать от вашего имени или получать данные о балансе, необходимо подключить API ключи вашей биржи.

## Безопасность прежде всего

Мы рекомендуем использовать **только** ключи с разрешениями на чтение и торговлю. **Никогда** не включайте разрешение на вывод средств (Withdrawal).

### Шаги подключения:

1. Перейдите в раздел **Настройки** -> **API Ключи**.
2. Нажмите кнопку **Добавить ключ**.
3. Выберите вашу биржу (например, Binance или Bybit).
4. Введите API Key и API Secret.
5. Протестируйте соединение с помощью кнопки **Проверить**.

Если всё настроено верно, статус ключа изменится на "Активен".
